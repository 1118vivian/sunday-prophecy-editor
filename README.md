# 主日申言稿編輯器

這是一個可在瀏覽器使用的主日申言稿編輯器。

## 功能

- 填寫組別、姓名、週次、日期、題目、經文、啟示、應用
- 即時預覽主日申言稿版面
- 產生並下載 DOCX
- 產生並下載 PDF

## 使用方式

直接開啟 `index.html`，或部署到 GitHub Pages 後使用公開網址。

## GitHub Pages

部署到 GitHub 後，可在 repository 的 `Settings` -> `Pages` 中選擇：

- Source: `Deploy from a branch`
- Branch: `main`
- Folder: `/root`

儲存後 GitHub 會產生公開網址。
